# SQL Queries 

This folder contains all SQL queries used in the project to explore, clean, and analyze the sales dataset.
The queries are organized to reflect a real-world analytics workflow, moving from raw data understanding to advanced analysis.

- Structure & Purpose
Data Exploration
Initial review of tables and columns
Understanding data structure, key fields, and relationships

-Data Quality & Cleaning
Checking for missing values, duplicates, and invalid records
Validating primary and foreign key relationships
Ensuring data consistency before analysis

-Data Analysis
Sales and profit analysis over time
Customer-level and product-level performance
Regional and category-based analysis

-Advanced SQL
Joins: Combining sales, customer, and product tables for meaningful insights
Aggregations: Calculating totals, averages, and performance metrics
Window Functions: Ranking products and analyzing trends
Views: Creating reusable, analysis-ready datasets for reporting and Power BI

-Output
The SQL queries produce clean, analysis-ready datasets that are later used in:
Business insight generation
Power BI dashboards for visualization
Identifying growth trends, top-performing products, and loss-making segments
